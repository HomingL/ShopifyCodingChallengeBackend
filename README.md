# Image Repo

## App Description

Like the name of the project, this applicaiton provides a platform for users to store their images to the repository and browse their user's public images.  

### development
```bash
$ npm install
$ npm run watch
$ npm run dev
```

### test
```bash
$ npm install
$ npm run test
```
